# SalesForecast – интеллектуальный сервис прогнозирования продаж для МСБ

## О проекте
Веб-приложение на Streamlit, которое автоматически подбирает лучшую модель прогнозирования продаж. Пользователь загружает данные о продажах, а система тестирует 3 алгоритма и выбирает оптимальный по метрике MAPE.

## Стек технологий
- **Python**: pandas, scikit-learn, statsmodels, xgboost, numpy
- **Визуализация**: plotly, matplotlib
- **Интерфейс**: Streamlit
- **Метрики**: MAPE (Mean Absolute Percentage Error), RMSE (Root Mean Squared Error)

## Как работает система
1. Загрузка CSV-файла с историческими данными о продажах
2. Автоматическое обучение трёх моделей:
   - Holt-Vinters 
   - Random Forest 
   - XGBoost 
3. Расчёт MAPE и RMSE для каждой модели на тестовой выборке
4. Выбор победителя по MAPE и отрисовка прогноза на графике
